# Yuwei Hu Personal Website

This is a static personal academic website.

## How to use with GitHub Pages

1. Create a GitHub repository named `YuweiHu17.github.io` or use an existing GitHub Pages repository.
2. Upload `index.html` and the `assets/` folder to the root of the repository.
3. In GitHub, go to Settings > Pages and select the main branch.
4. The site will be available at `https://YuweiHu17.github.io/`.

To edit text, open `index.html` and change the corresponding section.
